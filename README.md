## How to run

1. [Install Docker](https://docs.docker.com/engine/install/)
2. Install OS-specific MySQL driver, for macOS use `homebrew`:

   ```bash
   brew install mysql
   ```

3. Run docker compose at project root

   ```bash
   docker compose up
   ```

4. Open [localhost:8000](localhost:8000) in browser
